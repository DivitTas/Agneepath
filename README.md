# Agneepath

A Unity game built for the **Campfire Hack Club Game Jam**. Agneepath is a 2D platformer & action game.  

---

Made in a short timeframe for the Campfire Hack Club Game Jam, this project was all about learning, prototyping, and getting something *shippable* under deadline.
## Made by:
- [Divit](https://github.com/DivitTas) – Gameplay systems, core mechanics  
- [Sarika](https://github.com/sarikacoder-art) - Sprite art and animation.
- [Sarthak](https://github.com/sarthakm008) - Level design & Cutscenes
